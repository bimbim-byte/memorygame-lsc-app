## Installation
- Download the latest exe release: https://github.com/bimbim-byte/memorygame-lsc-app/releases/latest
- Create a new folder anywhere, then move the exe file into that folder
- Run the application
- Once opened, click the "Create&Check Config" button; this will create 3 folders and 2 new files
- Install Lightshot, then set the screenshot location to the "input" folder

## How to Use
- Play Lost Saga Matching Pair using the provided screenshot patterns
- Once all screenshot patterns are captured, press the "Start Memory" button
- An overlay image will appear; click on the overlay image
- To go to the next overlay, press Alt+X on your keyboard; to go back to the previous overlay, press Alt+Z
- When finished, press the "Stop Memory" button; all images in the folder will be deleted, and you can start Matching Pair again
